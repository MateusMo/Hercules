# 🏛️ Hercules CLI

Hercules is a fun terminal-based C CLI tool that simulates file deletion with a cinematic animation inspired by the strength of Hercules.

Instead of silently removing files like `rm`, Hercules brings a dramatic visual experience to the terminal before performing the operation.

---

## ⚙️ Features

- Animated “Hercules strength” deletion sequence
- Multi-file support
- Colored terminal output
- Simple CLI interface
- Built in C with minimal dependencies

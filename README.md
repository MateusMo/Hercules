# 🏛️ Hercules CLI

Hercules is a fun terminal-based C CLI tool that simulates file operations with cinematic animations inspired by the strength of Hercules.

Instead of silently executing system commands like `rm` or `find`, Hercules brings a dramatic visual experience to the terminal.

---

## ⚙️ Features

- 🎬 Animated Hercules-style sequences
- 🧹 Safe file removal simulation (`--clear`)
- 🔍 Deep system search (`--find`)
- 🎨 Colored terminal output with ANSI effects
- 📂 Multi-file support for deletion
- ⚡ Lightweight C implementation with minimal dependencies
